# Kraken Sanctions Compliance Dashboard

**Interview Demo** for **Sr. Compliance Associate, Sanctions** role at Kraken.

This interactive dashboard showcases my understanding of sanctions compliance workflows, regulatory reporting obligations, and automation opportunities — directly aligned with the responsibilities outlined in the job description.

---

## 🎯 Purpose

Demonstrates how I would support the **Chief Sanctions Officer** by providing:
- Real-time visibility into regulatory reporting status
- Executive-level program health metrics
- Crypto-specific sanctions risk monitoring
- Simulated AI-assisted narrative generation (Claude-style)
- Ability to load and screen against real OFAC SDN data

---

## ✨ Key Features

- **Executive Overview** — High-level KPIs and sanctions landscape
- **Regulatory Reporting Hub** — OFAC & international filing tracker with one-click AI narrative generation
- **Sanctions Intelligence** — Quick screening tool + crypto exposure metrics
- **Real OFAC Integration** — Upload official SDN.csv for live entity screening
- Clean, minimal Apple-inspired design optimized for senior leadership

---

## 🛠️ Tech Stack

- **Framework**: Streamlit (Python)
- **Visualization**: Plotly
- **Data Handling**: Pandas
- **Styling**: Custom Apple-system fonts + clean Tailwind-inspired design
- **Deployment**: Streamlit Community Cloud
